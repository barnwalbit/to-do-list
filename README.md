# to-do-list
A responsive to-do web app with project management, priority filters, due dates, and local storage persistence. Built with HTML, CSS and JavaScript.

^ Features
- Add, edit, and delete tasks
- Priority levels (High, Medium, Low)
- Project organisation
- Due date tracking
- Local storage persistence (data get saved in browser)
- Fully responsive and mobile friendly

^ Live Demo
https://barnwalbit.github.io/to-do-list

^ Tech Stack
- HTML
- CSS
- JavaScript
